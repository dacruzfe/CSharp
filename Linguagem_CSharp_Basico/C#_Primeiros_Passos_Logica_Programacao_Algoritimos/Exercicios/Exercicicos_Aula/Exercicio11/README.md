## Exercicio Calcular numeros
Faça um programa que leia N números inteiros e armazene-os em um vetor. Em seguida, mostrar na tela todos os números negativos lidos.

| Exemplos de Entrada | 	Exemplos de Saída  |
| ------------- | ------------- |
| 6  | -2 |
| 8 -2  9 10 -3 -7 | -3 |
|   | -7 |
